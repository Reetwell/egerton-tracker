# Claude Code - Slash Commands Guide

A plain English guide for using Claude Code's slash commands. No technical experience needed!

---

## What Are Slash Commands?

Slash commands are shortcuts you type in Claude Code that start with a forward slash `/`. They help you do things quickly without typing long instructions. Just type the command and press Enter.

---

## Getting Help

| Command | What It Does |
|---------|--------------|
| `/help` | Shows you how to use Claude Code. Start here if you're new! |
| `/release-notes` | See what's new in the latest version of Claude Code |
| `/doctor` | Checks if Claude Code is working properly. Use this if something seems broken |
| `/bug` | Report a problem directly to the team who makes Claude Code |

---

## Managing Your Conversation

| Command | What It Does |
|---------|--------------|
| `/clear` | Wipes the conversation clean so you can start fresh |
| `/rewind` | Go back to an earlier point in your conversation (like an undo button) |
| `/resume` | Pick up where you left off in a previous conversation |
| `/export` | Save your conversation to a file or copy it to your clipboard |
| `/compact` | Shrinks your conversation to use less memory. Helpful for long chats |

---

## Changing AI Models

| Command | What It Does |
|---------|--------------|
| `/model` | Switch between different AI models (like Sonnet, Opus, or Haiku). Each has different strengths - Opus is smartest but slower, Haiku is fastest but simpler |

---

## Checking Your Usage

| Command | What It Does |
|---------|--------------|
| `/context` | Shows a visual grid of how much memory your conversation is using |
| `/cost` | See how many tokens (AI credits) you've used and what it costs |
| `/usage` | Check your plan limits - how much you can use per day/month |

## Settings and Configuration

| Command | What It Does |
|---------|--------------|
| `/config` | Opens the main settings menu where you can change options |

| `/permissions` | Control what Claude is allowed to do (like editing files) |
| `/privacy-settings` | Manage how your data is stored and handled |

## Project Tools

| `/review` | Ask Claude to review your code and give feedback |

## Quick Start Tips

### If you're brand new:
1. Type `/help` to see the basics
2. Type `/doctor` to make sure everything is working
3. Type `/status` to see your account info

### If your conversation gets long:
1. Use `/compact` to shrink it down
2. Or use `/clear` to start fresh

### If you want to save your work:
1. Use `/export` to save the conversation
2. Use `/memory` to save important project info for next time

### If you're working on code:
1. Use `/init` at the start of a project to help Claude understand it
2. Use `/review` to get feedback on your code
3. Use `/todos` to track what needs doing

### If you hit your usage limits:
1. Use `/usage` to check your limits
2. Use `/cost` to see what you've used
3. Consider using `/model` to switch to a cheaper model (like Haiku)

---

## How to Use a Slash Command

1. Click in the Claude Code input area
2. Type the command (e.g., `/help`)
3. Press Enter
4. Follow any prompts that appear

That's it! Slash commands make Claude Code faster and easier to use.
